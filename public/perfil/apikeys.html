Puedes utilizar tu API key para acceder a nuestros servicios programáticamente. Aquí tienes un ejemplo:

// Ejemplo de uso con JavaScript
const options = {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer sk_live_3x5d8d7f...'

  })
